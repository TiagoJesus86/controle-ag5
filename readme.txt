Como compilar

Para compilar e gerar os executáveis você precisará de algumas ferramentas. 

Embora o "Controle - Agenda 5" possua o código fonte aberto, 
algumas ferramentas usadas no desenvolvimento são comerciais e você deve 
adquirir licenças dessas ferramentas para compilar.

Lista de ferramentas necessários :

- Embarcadero Delphi 10.3.2 ( Rio ) - Enterprise Edition
- JCL
- JVCL
- TMS Component Pack
- TMS FlexCel
- QuickReport 6.00
- SMImport
- SMExport
- TurboCode - Pcks
- Firebird 2.5.7

Assim que tiver as ferramentas prontas e instaladas, 
abra o arquivo de grupo de projeto "ControleAg5\Projects\CL.Ag5X_RIO.groupproj" e compile.
